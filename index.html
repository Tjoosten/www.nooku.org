For like-minded developers

Joomla extension builders, integrators and hackers alike

Nooku is installed in Joomla as a plugin and lets you focus on the "business" logic of your extension.

By doing most of the repetitive work for you, boilerplate code is eliminated which greatly reduces the development time.

The design pattern based architecture makes your extension more flexible, re-usable, and easily extensible.

Nooku abstracts the differences between Joomla versions for you. Code it once, use it everywhere.
