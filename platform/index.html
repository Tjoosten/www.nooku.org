What is Nooku Platform?

Nooku Platform was born at the end of 2010 with the goal to create a leaner and meaner multi-site and multi-lingual distro of Joomla 1.5 for the Belgian Police. Originally named Nooku Server and renamed to Nooku Platform in 2014.

Compatibility with Joomla was dropped soon after Joomla 1.5 reached end of life in 2012. Since then, development continued in an effort to completely refactor the code base to a modern and lean architecture using well-established coding standards and best practices.

At the moment Nooku Platform is at version 0.9 and work is ongoing to remove the last bits of Joomla legacy code. Once this work is completed a 1.0 alpha release will be made available.

Be forewarned!

Not for the faint of heart

Nooku Platform is designed for developers and early adopters, and can sometimes break down completely.

Nightly updates

Nooku Platform changes almost every day. Check the develop branch for the latest and greatest.
