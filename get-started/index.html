Start developing with Nooku

Follow these quick steps and start building Joomla extensions in minutes.

1. Install Nooku Framework

Go to the root directory of your Joomla installation on the command line and execute these commands:

{% highlight bash %} $ [ -f composer.json ] && composer update $ composer require nooku/nooku-framework:2.* {% endhighlight %}
